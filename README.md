# 📚 Digital Recipe Books Application

A full-stack web application for managing digital recipe books, allowing users to organize, search, and secure their personal recipes in a structured and intuitive way.

---

## 🚀 Project Overview

This application enables users to create and manage digital recipe books, categorize recipes, and easily filter and search through their collection.

The system focuses on:
- Clean organization of recipes into books and indexes
- Secure user authentication
- Smooth and responsive user experience

---

## 🛠️ Tech Stack

### Backend
- Java
- Spring Boot
- Spring Security
- JWT Authentication
- REST API

### Frontend
- React
- Material UI (MUI)
- Axios

### Database
- H2 In-Memory Database (H2 Console)

---

## 🔐 Security

- JWT Token-based authentication
- Secure API endpoints per user
- Protected routes for authorized access only

---

## 📁 Features

- User registration and login
- JWT-secured session management
- Create and manage recipe books
- Add, edit, delete recipes
- Organize recipes into categories / indexes
- Search and filtering system
- Responsive UI with Material Design
